## 산술 연산자

| + | 더하기 |
| --- | --- |
| - | 빼기 |
| * | 곱하기 |
| / | 나누기 |

| // | 나누기(몫) |
| --- | --- |
| % | 나머지 값 |
| ** | 제곱 |

```python
num1=100
num2=50
print(num1+num2) #덧셈
print(num1-num2) #뺼셈
print(num1*num2) #곱셈
print(num1/num2) #나눗셈

print(102//20)
print(102%20)
print(2**3)
```

```
150
50
5000
2

5
2
8
```

### 복합 대입 연산자

복합 대입 연산자: 변수에 값을 변경한 후에 다시 자신에게 대입하는 역할

| += | num=num+n |
| --- | --- |
| -= | num=num-n |
| *= | num=num*n |
| /= | num=num/n |
| //= | num=num//n |
| %= | num=num%n |
| **= | num=num**n |

```python
num=20
num+=3 ; print(num)
num-=3 ; print(num)
num*=3 ; print(num)
num/=3 ; print(num)
num//=3 ; print(num)
```

```
23
20
60
20
6
```

## 비교 연산자와 논리 연산자

**비교 연산자**는 어떤 것이 큰지, 작은지, 같은지를 비교하는 것으로 그 결과는 참(True)이나 거짓(False)이 된다.

| == | 같다 |
| --- | --- |
| ≠ | 같지 않다 |
| > | 크다 |
| < | 작다 |
| >= | 크거나 같다 |
| <= | 작거나 같다 |

예시 1

```python
score=int(input('수학 시험 점수를 입력하시오:'))
print(score>=70)
```

```
수학 시험 점수를 입력하시오: 80
True

#70 이상의 값을 입력하면 True가 되고, 70 미만의 값을 입력하면 False가 됨
```

예시 2

```python
n1=100
n2=200

print(n1==n2)
print(n1>n2)
print(n1<=n2)
```

```
False
False
True
```

**논리 연산자**는 비교 연산자가 여러 번 필요할 때 사용한다. 예를 들어 num 변수의 값이 10과 20 사이에 있어야 한다는 ‘num은 10보다 커야 한다’ 그리고 ’num은 20보다 작아야한다’ 이다. 이럴 때 비교 연산자를 사용한다.

| and | 그리고 | 둘 다 참이어야 참 |
| --- | --- | --- |
| or | 또는 | 둘 중 하나만 참이어도 참 |
| not | 부정 | 참이면 거짓, 거짓이면 참 |

| AND |  |  |
| --- | --- | --- |
| x | y | 결과 |
| T | T | T |
| T | F | F |
| F | T | F |
| F | F | F |

| x | y | 결과 |
| --- | --- | --- |
| T | T | T |
| T | F | F |
| F | T | F |
| F | F | F |

| x | y | 결과 |
| --- | --- | --- |
| T | T | T |
| T | F | F |
| F | T | F |
| F | F | F |
