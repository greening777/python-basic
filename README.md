# 🐍 기초 파이썬 학습 자료

이 레포지토리는 **기초 파이썬 수업**에서 배운 내용을 정리하고 공유하기 위해 만들어졌습니다.  
파이썬을 처음 배우는 초보자를 대상으로 기본 개념부터 예제 코드까지 담고 있습니다.
